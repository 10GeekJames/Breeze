========================
Breeze  Todo: an example 
========================

Make sure you have the current copy of breeze reference files
1] ThirdParty/Breeze.WebApi - the breeze server assembly for accessing a Web API
2] ThirdParty/Irony - breeze uses this language lib to analyze query strings (http://irony.codeplex.com/)
3] client/lib/breeze.debug.js - the unminified breeze javascript library

You may also want to acquire the current versions of the other 
3rd party javascript libraries in client/lib.
