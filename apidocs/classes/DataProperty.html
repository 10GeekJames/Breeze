DataProperty Class

Defined in: entityMetadata.js:1244
Module: entityModel

A DataProperty describes the metadata for a single property of an EntityType that contains simple data.

Instances of the DataProperty class are constructed automatically during Metadata retrieval. It should almost never be necessary to create one directly.
