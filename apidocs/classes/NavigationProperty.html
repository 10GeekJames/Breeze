NavigationProperty Class

Defined in: entityMetadata.js:1412
Module: entityModel

A NavigationProperty describes the metadata for a single property of an EntityType that return instances of other EntityTypes.

Instances of the NavigationProperty class are constructed automatically during Metadata retrieval. It should almost never be necessary to create one directly.
